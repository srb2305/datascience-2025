Pandas is a high-level data manipulation tool built on top of NumPy. It offers data structures like Series and DataFrame that are ideal for working with tabular or labeled data (like Excel sheets, SQL tables, etc.).

âž¤ Core Data Structures:
Series: 1D labeled array (like a single column)

DataFrame: 2D labeled data structure (like a full table)

âž¤ Common Use-Cases:
Data Cleaning

Data Transformation

Filtering, sorting, grouping

Import/export CSV, Excel, JSON, SQL

Time series manipulation

âž¤ Example:
import pandas as pd

# Creating a DataFrame
data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35]
}

df = pd.DataFrame(data)
print(df)

# Filter rows where Age > 28
print(df[df['Age'] > 28])
